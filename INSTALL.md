# Инструкция по установке и запуску

## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создание тестовых данных (опционально)
```bash
python init_data.py
```

### 3. Запуск приложения
```bash
python main.py
```

### 4. Открыть в браузере
```
http://localhost:5000
```

## Тестовые аккаунты

## Структура проекта для курсовой работы

### Техническая часть:
1. **Backend:** Flask (Python веб-фреймворк)
2. **База данных:** SQLite с ORM SQLAlchemy
3. **Frontend:** HTML5, CSS3, Bootstrap 5
4. **Аутентификация:** Flask-Login
5. **Безопасность:** Хеширование паролей

### Функциональные требования:
- ✅ Регистрация и авторизация пользователей
- ✅ Отправка и получение сообщений
- ✅ Система приоритетов сообщений
- ✅ Отметка сообщений как прочитанных
- ✅ Панель управления с статистикой
- ✅ Адаптация под медицинскую сферу

### Особенности для здравоохранения:
- Специализированные отделения (кардиология, неврология и т.д.)
- Медицинские должности (врач, медсестра, заведующий)
- Система приоритетов для экстренных случаев
- Конфиденциальность медицинской информации

## Демонстрация работы

### Сценарий 1: Обычное сообщение
1. Войти как `ivanov_doctor`
2. Отправить сообщение медсестре
3. Войти как `petrov_nurse`
4. Просмотреть и отметить сообщение как прочитанное

### Сценарий 2: Экстренное сообщение
1. Войти как `sidorova_head`
2. Отправить экстренное сообщение врачу
3. Войти как `ivanov_doctor`
4. Увидеть выделенное красным экстренное сообщение

### Сценарий 3: Панель управления
1. Войти в систему
2. Просмотреть статистику на главной странице
3. Использовать быстрые действия
